## My Time


Clockify Client App to Record the Time Entering the Task.


Technologies & Tools that are used in this projects are:

    Flutter
    Flutter Bloc
    Clean Architecture
    Dependency Injection
    Background Services
    Local Notification
    Chopper Http client generator
    ObjectBox Flutter Database
